🛒 Cadastro de Produtos 🚀

📌 Descrição do Projeto
Este projeto é uma aplicação web simples desenvolvida com HTML, CSS e JavaScript.
A aplicação permite que o usuário cadastre produtos informando nome, preço e quantidade em estoque. Após preencher os campos e clicar no botão Cadastrar, os dados são validados e exibidos automaticamente na tela.
O sistema utiliza uma lista (Array) para armazenar os dados e também um Set, garantindo a prática do conceito de remoção de valores repetidos.

🎯 Objetivo
O objetivo deste projeto é praticar conceitos básicos do desenvolvimento web, como:
Estruturação de páginas com HTML
Estilização com CSS
Captura de dados através de inputs
Validação de campos obrigatórios
Manipulação do DOM com JavaScript
Uso de Arrays e Set
Exibição dinâmica de informações sem recarregar a página

✅ Funcionalidades
📌 O sistema permite:
Inserir nome do produto
Inserir preço
Inserir quantidade em estoque
Validar se os campos foram preenchidos corretamente
Armazenar os dados em uma lista (Array)
Utilizar Set para remover valores repetidos
Exibir os dados cadastrados na tela automaticamente

🛠 Tecnologias Utilizadas
HTML5
CSS3
JavaScript

⚙️ Como Funciona
Ao clicar no botão Cadastrar, o sistema:
Captura os valores digitados pelo usuário
Verifica se os campos estão preenchidos corretamente
Armazena os dados em um Array
Cria um Set com os dados para evitar repetições
Exibe as informações cadastradas diretamente na página
